O **Manu Coder** é um agente de Inteligência Artificial criado para ajudar estudantes no aprendizado de **Python**.
Ele auxilia no desenvolvimento, explicação de códigos e prática durante os estudos.

---

## 🚀 Funcionalidades

* 📘 Auxílio no aprendizado de Python
* 💡 Explicação de códigos
* 🧠 Suporte para exercícios e dúvidas
* ⚡ Interface simples com Streamlit

---

## 🛠️ Tecnologias utilizadas

* Python 3.13
* Streamlit
* Bibliotecas listadas em `requirements.txt`

---

## ⚙️ Como executar o projeto

### 1. Criar ambiente virtual

```bash
conda create -name manu_Coder python=3.13
```

---

### 2. Ativar o ambiente

```bash
conda activate manu_Coder
```

---

### 3. Instalar dependências

```bash
conda install pip
pip install -r requirements.txt
```

---

### 4. Executar a aplicação

```bash
streamlit run manu_coder.py
```

---

## 🧹 Encerrar e remover ambiente (opcional)

```bash
conda deactivate
conda remove --name manu_Coder --all
```

---

## 🔐 Segurança

⚠️ Nunca exponha chaves de API no código.
Utilize um arquivo `.env` para armazenar suas credenciais com segurança.
